## 1. R Square（R²，決定係數）
是什麼？
R² 表示模型對資料的解釋能力。它的值在 0 到 1 之間。

意思：
R² = 1：模型能完全解釋資料中的變異（理想狀況）。

R² = 0：模型完全無法解釋資料變異（等於亂猜）。

R² 越高 → 模型越能解釋目標變數的變化。

舉例：
假設你在做「廣告支出對銷售的影響」分析：

如果 R² = 0.85，表示 85% 的銷售變化可以用你的模型（例如：廣告花費）來解釋。
